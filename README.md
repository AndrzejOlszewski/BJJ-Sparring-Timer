# BJJ Sparring Timer 🥋

A simple Brazilian Jiu-Jitsu sparring round timer built with HTML, CSS, and JavaScript.

## About

The timer allows users to:
- Start a round
- Pause the timer
- Reset the timer back to 5 minutes

## Features

- 5-minute sparring round countdown
- Start, pause, and reset buttons
- Simple user interface
- Timer display updates every second

## Technologies Used

- HTML
- CSS
- JavaScript

While building this project I practised:

- Using `setInterval()` for countdowns
- Updating webpage content with JavaScript
